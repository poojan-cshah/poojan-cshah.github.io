## Logo Specifications

- **Format**: PNG with transparent background (preferred)
- **Size**: Minimum 200x200px (will be displayed at 64x64px)
- **Aspect ratio**: Square or nearly square works best
- **File naming**: Use lowercase, exact filenames as listed above

## Tips

- Use official brand assets when available
- Ensure you have permission to use the logos
- Keep file sizes reasonable (< 100KB each)
- PNG format works better than JPG for logos

## Testing

After adding logos, open `index.html` in your browser to verify they display correctly.
